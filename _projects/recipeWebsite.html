After building my own HTTP server I set out to launch a website on top of it. The site (which is actively being used) is a recipe management app which can be found here. The site was written as a "one page app" in JavaScript + JQuery. The JavaScript uses the Model-View-Controller design paradigm and the functional inheritance pattern (as apposed to prototypical inheritance) for defining objects. For the minimal stilling, I used Bootstrap. The site is currently being hosted on an EC2 micro instance.

The key feature of the site is that it can extract recipes from other recipe websites. The user, once finding a recipe they wish to save, can enter the URL into the recipe site to extract the recipe. This extraction is accomplished on the web server using a collection of data extraction scripts written in Emacs. This logic may eventually be re-written in JavaScript. The following is a list of sites currently supported:

  • Yummly
  • The Food Network
  • Better Homes And Gardens
  • All Recipes
  • Real Simple
  • Manjula's Kitchen
